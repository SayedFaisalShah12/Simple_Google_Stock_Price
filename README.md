# Google Stock Tracker (Streamlit App)

This Streamlit app displays Google's stock **closing prices** and **volume** using data from the Yahoo Finance API (`yfinance`).

## Features
- Fetch stock data for ticker: **GOOGL**
- Display:
  - Closing Price Chart
  - Volume Chart

## How to Run (Locally)
```bash
pip install -r requirements.txt
streamlit run main.py
